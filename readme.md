> The personal website of Neil Kistner.

## Setup

```sh
$ git clone https://github.com/wyze/wyze.github.io.git
$ cd wyze.github.io
$ yarn # or `npm install`
```

## Development

```sh
$ npm run dev
```

## Test

```sh
$ npm test
```

## Build

```sh
$ npm run build
```

## Production

```sh
$ npm start
```
